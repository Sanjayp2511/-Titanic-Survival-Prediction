# Titanic Survival Prediction 🚢

## Project Overview
Built machine learning models to predict survival 
of Titanic passengers based on features like 
age, sex, fare and passenger class.

## Results
| Model | Accuracy |
|---|---|
| Logistic Regression | 79.89% |
| Random Forest | 83.24% |

## Key Findings
- Fare was the most important survival factor (28.3%)
- Women had much higher survival rates than men
- First class passengers survived more than third class

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib
